## 项目简介

本仓库用于归档笔者在高三数学一对一与线上教学中沉淀的真实课堂板书、阶段复习材料与专题整理。内容强调完整推导、板书层次、通法模型与高考表达习惯，并自 2026-05 起将数学资源按**文件名关键词**归入与人教 A 版对齐的模块文件夹，便于检索与更新。
