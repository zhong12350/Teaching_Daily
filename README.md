## 项目简介

本仓库用于归档笔者在高三数学一对一与线上教学中沉淀的真实课堂板书、阶段复习材料与专题整理。内容强调完整推导、板书层次、通法模型与高考表达习惯，便于复盘教学组织方式，并持续扩展为可检索、可维护的体系化资料库。

近期增量主要包括：**导数专题**成套 Word 讲义（专题 01–38）、多份解析几何 / 立体几何 / 数列 / 三角函数专题 PDF、新高考与名校模拟卷相关资源，以及部分一轮复习学案与压缩包归档。

## 本地使用说明

- **仅浏览 PDF**：使用 GitHub 网页或 Pages 站点链接即可。
- **Word / 压缩包**：建议在终端执行 `git clone https://github.com/zhong12350/Teaching_Daily.git`，避免单文件下载遗漏或中文路径在浏览器中的编码问题。
- **站内「DeepSeek 总结」**：依赖 `assets/deepseek.js`，用于生成可复制的提示文案；详见 [`assets/README.md`](./assets/README.md)。

## 教学特点（与站点一致）

- **逻辑严谨**：重视定义、性质与推导链，减少经验性跳步。  
- **板书清晰**：强调层次与关键词落点，便于课后复盘。  
- **体系化**：以知识网络 + 通法模型组织，不停留于碎片刷题。  
- **高考导向**：对齐高频考点、规范表述与典型失分点。

## 适合人群

- **高三考生**：系统复习、专题突破与冲刺阶段查漏补缺。  
- **高中数学自学者**：需要框架感与「题型—方法」对照的读者。  
- **师范 / 家教同仁**：可参考板书组织与专题拆分方式（请注意版权与校内规定）。

## 项目亮点

- 真实课堂板书与成套专题讲义并存，兼顾「教学现场」与「可复用教案」。  
- 导数大专题单线贯穿，便于按专题排课。  
- 提供静态门户页 + 归档索引，利于对外展示与自我迭代。
