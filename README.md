# 高三数学系统教学资料

> 南方科技大学本科生 | 长期高三数学一对一/线上教学积累 | 面向高考

---

## 项目简介

本仓库收录笔者在大学期间从事高三数学教学的系统性资料，涵盖课堂板书扫描件、知识点梳理、题型方法总结及高考专项训练材料。

教学覆盖高三全年完整周期（2 月至 4 月为近期记录存档），资料体系化程度高，可供高三学生自学参考，亦可作为青年数学教师教学设计的参考素材。

---

## 教学理念与特点

| 维度 | 说明 |
|------|------|
| **逻辑严谨** | 注重推导过程的完整性，不跳步、不依赖直觉，强调从定义出发的严格论证 |
| **板书清晰** | 每节课手写板书结构化呈现，层次分明，便于复习回溯 |
| **通法优先** | 优先讲授普适解题方法，避免技巧依赖，培养迁移能力 |
| **高考导向** | 紧扣高考命题规律，精准把握考查重点与题型变化趋势 |
| **体系化思考** | 以知识网络为骨架，帮助学生建立完整的数学认知框架，而非碎片化刷题 |

## 关于作者

**南方科技大学**电子与电气工程系本科在读，长期利用课余时间从事高三数学一对一及线上教学。

教学目标明确：以严谨的数学思维、清晰的语言表达，帮助学生真正理解高中数学的本质，而非停留于题海训练。长期志向为进入深圳中学等顶尖高中担任数学教师，以高标准践行数学教育理念。
