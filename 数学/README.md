# 数学资料索引（人教 A 版高中数学知识点归类）

本目录下的子文件夹按**人民教育出版社《普通高中教科书·数学》（2019 版，人教 A 版）**的**知识模块**归类，便于按教材顺序检索；高三综合卷、成套真题等归入 **「12-高考真题与模拟卷」**。文件名本身已能反映主题，归类主要依据文件名中的关键词。

> **访问指引**
> - 在 **GitHub** 上浏览：打开 [仓库主页](https://github.com/zhong12350/Teaching_Daily)，进入 `数学/` 下对应子文件夹即可下载。
> - 在 **GitHub Pages** 上：请使用站点首页 [知识模块导航](../index.html#curriculum) 中的链接；单文件 PDF 可在首页「拓展资料」中直接打开部分条目。
> - **非数学**资料已移至仓库根目录 [`其他学科/`](../其他学科/)（如语文、生物零散文件）；**物理等**仍在 [`高三物理教学/`](../高三物理教学/)。

## 归类说明与局限

- 归类依据是**文件名中的显性关键词**，与教材章节的对应是**教学意义上的对齐**，并非严格「一题一章」。
- 跨章节明显的资料（如整轮「解析几何」打包）放在 **99** 或 **12**。
- 若某文件您认为应调整模块，可在本仓库中 **移动文件并修改本说明**，保持与 `index.html`、根目录 `README.md` 中的链接一致即可。
